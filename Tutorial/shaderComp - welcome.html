ShaderComp API The shader compilation and optimisation library

The ShaderComp API is a project whose purpose is to allow programmer to build shaders independently of the plateform the shader should be programmed for. That means that the shader creator should be able to generate the same shader for any platform allowed by the library without doind any change.

Moreover, an integrated plug-in system has been developped to allow any programmer to contribute to the developpment of the system. The more plugins it holds, the more polyvalent the API will become !

EPITA & Ubisoft Collaboration

ShaderComp is the result of the work of three student of the EPITA ingeneer school in Paris, collaborating with the Ubisoft France Corporation.

This has been done during the four last mounth of our fifth studying year, during wich we focused by curiosity on the graphics pipeline and what possibilities a dynamically programmable graphics pipeline can offer.

We are aware that many things still could be supported to let the library be as useful as the unity3D shaders plug-in for instance, but this project intend to be totally independant of the graphics engine it is generating the code for.

Made with python

We choose the language python to build our API for the power of it's high-level datatypes and the simplicity of writing, reading and debugging. It also allow easily to build a modular architechture needed to manade differents types of plug-in. The extendability of the library was one of the keypoint of the project.

Why use ShaderComp ?

Here are some keypoint of the project telling in a few words why this project has a great potential.

Easy-to-learn

Thanks to the use of the Python Programming language, the code is clear and concise. Moreover, there is a full User and Programmer tutorial available for you to introduce the API.

Multiplatform

The API can be used independently from any operating system for any platform if the python interpreter is installed and a Printer exists for this platform.

Simple

The use is explained in the User Tutorial: it's only a matter of netwoks drawing and building !

Growing

The project is designed in a very modular way under the MIT License. Feel free to read the Programmer tutorial and learn how to extend the features of the library !
