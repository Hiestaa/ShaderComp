Detailed Description

This package contains the class needed to print the code source of the shaders.

The JSONPrinter is use to export or import a project from a JSON file.

The GLSLPrinter is use to compute the GLSL Code of a project.

In this module, a printer is implemented (as an inheriting class from Printer) for each platform supported by the library.

The purpose of a printer is to run over the list of nodes of both boxes of the project, and to use for each node the right generator to append the source code of the shader to the buffer that will be written in the resulting files

By convention, it is assumed that for each shader and for each language, a generator can be found in this module in the folder that has the name of the shader. In this folder the generator is a class named [ShaderName][PrinterName].

For example, the generator of the shader CelShading, for the printer GLSLPrinter must be called CelShadingGLSLPrinter.
