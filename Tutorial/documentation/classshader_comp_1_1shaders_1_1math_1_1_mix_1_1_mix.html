Detailed Description

This shader can be used to mix two vectors of type float4 within a given factor.

Authors
Romain GUYOT de la HARDROUYERE
Matthieu BOURNAT
Antoine CHESNEAU
Version
0.1

2013-11-08
Type of this shader:
Fragment Shader

Input variables:

  • v1: vec4 is the first input vector
  • v2: vec4 is the second input vector
  • factor: float the mix factor between 0.0 and 1.0. If the factor is big the result will be near the vector v2. If the factor is low, the result will be near the vector v1.

Output variables:

  • result: float is the result of the operation

The resulting operation will be something like result = mix(v1, v2, factor)
